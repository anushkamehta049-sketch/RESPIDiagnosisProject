# Voice-Based Respiratory Disease Diagnosis Using Machine Learning

## Overview
This project predicts respiratory diseases from voice recordings using Machine Learning.

## Features
- Voice recording upload
- Audio preprocessing using Librosa
- Random Forest model for disease prediction
- Patient management system
- Report generation
- User-friendly web interface

## Technologies Used
- Python
- Flask
- Scikit-learn
- Random Forest
- Librosa
- HTML
- CSS
- JavaScript
- SQLite

## Installation

```bash
pip install -r requirements.txt
python app.py
```
